## Guide Resource

Hall Point Calculator provides hall applicants with a preliminary result of hall points they would likely receive under the Hall Point System. Applicants can input information following the sequences of questions in the Calculator to generate the hall point result. 

Applicants should understand that the hall point calculation results are estimates for reference only. The Student Housing and Residential Life Office is not responsible for the accuracy of projections in relation to the calculator.
